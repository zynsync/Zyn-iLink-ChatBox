# Zyn iLink ChatBox
-初中生个人开发-
📜 许可证: Apache 2.0 | 🤝 完整版权可买断 | 仅供学习交流使用

## ✨ 功能特性
- ✅ 微信官方iLink接口，扫码一键连接
- ✅ 支持文本、图片、语音、视频、文件收发
- ✅ 内置Web管理界面，手机电脑都能用
- ✅ AI自动回复，支持自定义系统提示词
- ✅ 媒体文件自动下载、转码和缓存
- ✅ 完美兼容Windows、Linux、Termux
- ✅ 零依赖复杂框架，单文件即可运行

## 🚀 快速开始
1. 克隆仓库
   ```bash
   git clone https://github.com/zynsync/Zyn-iLink-ChatBox.git
   cd Zyn-iLink-ChatBox
   ```
2.安装依赖
  ```bash
  pip install qrcode pycryptodome
  ```
3.运行
  ```bash
  python Zyn-iLink-ChatBox.py

  ```
