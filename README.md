## ✨ 功能特性

## 🚀 快速开始

### 1️⃣ 下载版本
去仓库的Releases下载脚本/下载启动器（推荐）

2️⃣ 安装依赖

```bash
pip install qrcode pycryptodome
```

3️⃣ 运行程序

```bash
python Zyn-iLink-ChatBox.py
```

💡 提示：
更推荐使用 启动器-带热更新 文件夹内的启动器！方便快捷 热更新

---

⚠️ 重要提醒

🔔 注意：对方扫码连接后，需要先让微信用户发一条消息获取token才能正常聊天！
